# MCP Stdio DB Assistant

[中文](./README.zh-CN.md) | [English](./README.en.md)

An MCP stdio database assistant built with FastAPI and OpenAI-compatible models.

This repository provides:

- an MCP stdio server for database tools
- a FastAPI chat gateway that lets an LLM call those tools
- a lightweight local setup for MySQL-based querying workflows
